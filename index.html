Random Test Mode

This version completely ignores real properties and generates random values for:

  • Serial number (9000–9999)
  • Start X/Y (±5000 around approximate bar center)
  • End X/Y (offset from start)

Purpose: Prove that single point markups work correctly in your extension.

Labels will appear near the selected rebar (using bounding box centers + small offset).
